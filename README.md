**CineMood** è un'esperienza cinematografica immersiva che trasforma le tue emozioni in visioni. Non è il solito motore di ricerca: è un curatore d'atmosfere basato sull'intelligenza artificiale che comprende il tuo stato d'animo e ti guida verso il film perfetto per la tua serata.

---

## ✨ L'Esperienza

Dimentica lo scrolling infinito tra i cataloghi. Con CineMood, il viaggio inizia da te.

- **🧠 AI-Driven Empathy**: Alimentato da **Gemini 3 Flash**, CineMood analizza descrizioni complesse di stati d'animo, desideri o semplici suggestioni per estrarre l'essenza di ciò che vuoi vivere.
- **🎙️ Voice-to-Vision**: Non hai voglia di scrivere? Esprimi il tuo mood a voce. L'integrazione del riconoscimento vocale rende l'interazione fluida e naturale.
- **🌈 Immersive UI**: Un'interfaccia "Atmospheric" con gradienti dinamici, effetti di grana cinematografica e scanline che reagiscono al tuo tocco.
- **🎨 Personalizzazione Totale**: Scegli la tua palette cromatica (Arancione, Blu, Verde, Viola, Rosso). L'app ricorda la tua preferenza grazie alla persistenza locale.
- **📱 Social Sharing**: Hai trovato la visione perfetta? Condividi i tuoi consigli personalizzati direttamente su WhatsApp con un click.

---

## 🚀 Funzionalità Principali

- **Analisi del Mood**: Inserisci testi come *"Una serata piovosa, solitudine riflessiva, atmosfere sci-fi alla Blade Runner"* e ottieni consigli mirati.
- **Streaming Intelligence**: I film suggeriti includono la piattaforma dove sono disponibili (Netflix, Prime Video, Disney+, MUBI, ecc.) e un link diretto.
- **Deep Dive**: Ogni consiglio include trama, "perché guardarlo" (una frase evocativa personalizzata), poster e valutazioni della critica (IMDb, Rotten Tomatoes).
- **Refine Search**: Aggiungi dettagli al tuo mood attuale per affinare i risultati senza ricominciare da zero.

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS (Mobile-first, Fluid Design)
- **Animations**: Motion (framer-motion)
- **AI Engine**: Google Generative AI SDK (@google/genai)
- **Icons**: Lucide React

## 📦 Installazione Locale

1. Clona il repository:
   ```bash
   git clone https://github.com/tuo-username/cinemood.git
   ```
2. Installa le dipendenze:
   ```bash
   npm install
   ```
3. Configura la tua API Key di Gemini in un file `.env`:
   ```env
   GEMINI_API_KEY=la_tua_chiave_qui
   ```
4. Avvia l'app:
   ```bash
   npm run dev
   ```
